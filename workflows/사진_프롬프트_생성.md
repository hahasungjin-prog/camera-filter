# Workflow: 사진 → 필름 감성 프롬프트 + 노션 저장

## 목표
사진 한 장을 AI에게 보내면:
1. 어울리는 필름 종류와 감성을 자동 분석
2. Midjourney / DALL-E 양쪽에서 쓸 수 있는 영어 프롬프트 생성
3. 인스타그램 훅 도입문 + CTA 문장 생성
4. 노션에 자동 저장

---

## 준비사항 (처음 한 번만)

### 패키지 설치
```
cd "j:\클로드 작업\카메라필터"
pip install -r requirements.txt
```

### .env 설정 확인
`.env` 파일에 3가지가 채워져 있어야 함:
- `ANTHROPIC_API_KEY` — Claude API 키
- `NOTION_TOKEN` — 노션 인테그레이션 토큰
- `NOTION_DATABASE_ID` — 결과를 저장할 노션 DB ID

**노션 DB ID 바꾸는 법:**
1. 노션에서 새 데이터베이스 페이지를 열기
2. 주소창 URL: `notion.so/워크스페이스/[32자리숫자]-?v=...`
3. 32자리 부분을 복사해서 `.env`의 `NOTION_DATABASE_ID`에 붙여넣기
4. 노션 인테그레이션을 해당 DB에 연결 (DB 페이지 우측 상단 `...` → `연결` → 내 인테그레이션 선택)

## 출력 결과

### 터미널에서 바로 확인
- 필름 유형, 감성, 감지된 기법
- 범용 프롬프트 (Midjourney / DALL-E 모두 사용 가능)
- Midjourney 전용 프롬프트 (`--ar 4:5 --v 6.1` 포함)
- DALL-E / ChatGPT 전용 프롬프트
- 인스타그램 훅 도입문
- CTA 문장

### 로컬 파일 (`output/` 폴더)
- `[사진이름]_result.txt` — 위 내용 전체를 텍스트로 저장

### 노션
- 새 페이지 자동 생성: `[날짜] [파일명] — [필름 유형]`
- 섹션 구성:
  - 📷 사진 분석 (필름/감성/기법)
  - ✨ AI 이미지 프롬프트 (3가지 버전)
  - 📱 인스타그램 콘텐츠 (훅 + CTA + 전체 캡션)

## 활용 흐름 (이후 작업)

```
run.py 실행
    ↓
노션에서 프롬프트 확인
    ↓
Midjourney 또는 ChatGPT에 프롬프트 복사-붙여넣기
    ↓
AI 이미지 생성
    ↓
노션의 훅/CTA/캡션을 인스타그램에 활용
    ↓
카드뉴스 or 릴스 제작 (카드뉴스 프로젝트 연동)
```

## 문제 해결

| 에러 | 원인 | 해결 |
|------|------|------|
| `ANTHROPIC_API_KEY 없음` | .env 미설정 | .env 파일 확인 |
| `NOTION_DATABASE_ID 없음` | .env 미설정 | 노션 DB ID 복사해서 .env에 붙여넣기 |
| `파일을 찾을 수 없습니다` | 경로 오류 | 이미지 경로 다시 확인 (따옴표로 감싸기) |
| `notion_client` 없음 | 패키지 미설치 | `pip install -r requirements.txt` |
| 노션 업로드 실패 | 인테그레이션 미연결 | 노션 DB에 인테그레이션 연결 확인 |
